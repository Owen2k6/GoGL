PRISM OS

About

Prism OS is a small open-source GUI based operating system written in C# with the Cosmos "Operating System Development Kit". It was created on May 11th, 2021 1:26AM UTC-8, by developer, and owner, terminal-cs under the original name "Lemon OS" as a fun side-project without any intention of it becoming anything greater. New features and updates are constantly rolled out; some features include: basic window manager, mouse system and a basic networking configuration.

Prism OS also has a small, growing, discord community where you can meet new people, contributors and the devs themselves! Join or share the server today using this link: https://discord.gg/DqNckqagMs
