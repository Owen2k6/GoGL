PRISM OS

About

Prism OS is a small open-source project created on May 11th, 2021 1:26AM UTC-8. Prism OS is made possible by the Cosmos open source project alongside all the developers and community based around it. Prism OS uses the developer kit which is based in C# to make a GUI based OS with a basic window manager and mouse system. The OS also contains a basic networking configuration (although it's very buggy) alongside a basic file system.
